**PhotoCollection**

Десктопный менеджер фотоколлекций с графическим интерфейсом, разработанный на Python. Приложение позволяет систематизировать личные фотографии, хранящиеся на компьютере, с помощью альбомов и тегов.

_Особенности:_

    Полностью локальное хранение: Все данные остаются на вашем компьютере.

    Гибкая организация: Создание альбомов и назначение тегов.

_Библиотеки:_

    Python 3 — основной язык программирования

    PyQt6 — современный фреймворк для создания графического интерфейса

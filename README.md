# PhotoCollection
PhotoCollection - это приложение для удобного хранения, организации и работы с фотографиями.
В приложении вы сможете создавать альбомы, редактировать названия альбомов, добавлять фотографии в альбомы, 
удалять альбомы и фотографии. Также в приложении есть возможность экспортировать данные о фотографиях в 
текстовый формат.


ИНСТРУКЦИЯ:

Создание альбома:
1. Нажмите кнопку "Создать альбом".
2. Введите название альбома в поле "Название альбома".
3. Нажмите кнопку "Создать".

Удаление альбома:
1. Выберите "Удалить альбом" из меню "Альбомы".
2. Выберите альбом для удаления.
3. Нажмите кнопку "Удалить".

Изменение альбомов:
1. Выберите "Изменить название альбома" из меню "Альбомы".
2. Выберите альбом из списка, который хотите изменить.
3. Введите новое название альбома в поле "Название альбома".
4. Нажмите кнопку "ОК".
 

Создание тегов:
1. Нажмите кнопку "Создать тег" из меню "Теги".
2. Введите название для тега в поле "Название тега".
3. Нажмите кнопку "ОК".

Удалить тег:
1. Выберите "Удалить тег" из меню "Теги".
2. Выберите тег для удаления.
3. Нажмите кнопку "Удалить".

Изменить тег:
1. Выберите "Изменить тег" из меню "Теги".
2. Выберите тег для изменения.
3. Введите новое название тега в поле "Название тега".
4. Нажмите кнопку "ОК".


Добавление изображения в альбом:
1. Нажмите кнопку "Добавить изображения".
2. Выберите альбом из списка.
3. Введите название к изображению.
4. Нажмите кнопку "Добавить".
5. Выберите изображение.

Удаление изображения:
1. Выберите "Удалить изображение" из меню "Изображения".
2. Нажмите на кнопку "Обновить альбомы"
3. Выберите альбом из которого хотите удалить изображение.
4. Нажмите на кнопку "Обновить изображения"
5. Выберите фотографию которую, хотите удалить.
6. Нажмите на кнопку "Удалить".

Изменить изображение:
1. Выберите "Изменить изображение" из меню "Изображения".
2. Нажмите на кнопку "Обновить альбомы"
3. Выберите альбом из которого хотите изменить изображение.
4. Нажмите на кнопку "Обновить изображения"
5. Выберите фотографию которую, хотите изменить.
6. Введите новое название фотографии в поле "Введите название:".
7. Нажмите на кнопку "Изменить".

Вывод изображения на главный экран:
1. Нажмите на кнопку "Обновить альбомы".
2. Выберите альбом из которого хотите вывести изображение.
3. Нажмите на кнопку "Обновить изображения".
4. Выберите фотографию которую, хотите вывести на главный экран.
5. Нажмите на кнопку "ВЫБРАТЬ".

Примечание:
В Photo Collection есть функция тегов которая позволяет удобнее организовать фотографии, но в данной версии теги не 
реализованны.

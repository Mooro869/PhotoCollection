# PhotoCollection
PhotoCollection - это приложение для удобного хранения, организации и работы с фотографиями.
В приложении вы сможете создавать альбомы, редактировать названия альбомов, добавлять фотографии в альбомы, 
удалять альбомы и фотографии. Также в приложении есть возможность экспортировать данные о фотографиях в 
текстовый формат.
